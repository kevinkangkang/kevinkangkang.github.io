我和我的祖国 (2019)      2020-05-27 / 爱国 感动 / ★★★★★ 推荐
之前太忙，到现在才看。
该电影一共讲述了7个故事。我就想说说其中的两个故事。
《前夜》讲述了在开国大典前夜，举国上下都在准备着，为的是能在十一能绝对不出错。林治远是此次自动升旗杆的设计者，他负责严谨的态度其实也是中国工程师们对待科学的态度，每一个步骤都必须考虑周全仔细，而不是应付式完成。在高潮部分，由于旗杆停滞装置坏了，需要临时征集材料当场制作新的装置，附近周围的人都资源捐出自己认为可以的金属材料，都想为此次举国大典贡献出自己的一份力量，就像影片中林治远说“两千万人的牺牲，换来的一面国旗，我们能做什么，绝不出错”，中国人民等这一天等得太久了。这时已是十一凌晨4、5点，天还是黑的，但是百姓们前往大典筹备处时举着的纸灯笼，就像是黑夜中的明灯，正是这些明灯才为黑暗的中国带来的光明，才铸就了现在的五星红旗。
《回归》讲述的是香港回归。当英国人说1984年签订的《中英联合声明》没有写明香港应该于7月1日的零时零分零秒回归，安文彬为代表的谈判组以坚定的态度表示香港必须在这一刻回归，这其实反映了香港回归刻不容缓，以及祖国日渐强大的自信。以朱涛为首的升旗手团队，为了这一次升旗，练了不下5000遍，就是为了能够做到在四十五秒准确地将国旗升起至旗杆顶端，每天睡觉都会想升旗的事情，害怕出错，压力十分巨大，据说当时他从6月30日就开始流鼻血，为了不影响任务而在鼻腔深处塞了棉花。在1997年7月1日0时0分0秒，《义勇军进行曲》正式响奏，全球的人都在见证着这一伟大时刻，国旗缓缓升旗。国歌奏闭，国旗至顶，全香港几百万人一起唱起《东方之珠》，以华哥为代表的普通群众看到这一刻，热泪盈眶，以莲姐为代表的港警在此时此刻更换为中国警徽，香港终于终于回归了。希望能够早日见证台湾回归！
还有许多历史故事都值得我们铭记，我和我的祖国正日渐强大。
工作细胞 はたらく細胞 (2018)     2020-04-30 / 拟人 治愈 / ★★★★☆ 推荐
脑洞是真的大。
将各种人体系统用拟人化的方式讲解，感觉挺有趣的hhh，比如红细胞脱核，就是把帽子上的小球摘下来（当时看到这个觉得那个小球有点cute），还有需要通过笑分泌的化学素来激活NT细胞，从而抑制癌细胞，真的不得不佩服这脑洞。
还是听推荐的，可以看到各种生物知识，让我想起高中学生物的感觉。有点怀念hhh。
（P.S. 白血球是真的是又可靠又温柔。
僵尸世界大战 / World War Z (2013)     2020-04-24 / 灾难 丧尸 / ★★★☆☆ 还行
感觉一般吧，剧情简单。
有两个地方印象比较深刻的是，耶路撒冷那全部人一高声歌唱就沦陷了，讽刺这些宗教的愚昧；还有一个地方就是男主生死未卜，家人就被送去难民区，也体现了政府的冷血。
但总体特效和演员的演技都还是很不错的，整个电影下来，紧张的地方和放松的地方都还可以，特别是逃跑耶路撒冷、炸飞机求生和科学室拿病毒的场景都让会让人看的很紧张。
电子竞技在中国 (2020)     2020-04-23 / 记录 电子竞技 中国 / ★★★★★ 力荐
还是挺不错的哈哈哈，从历史、经济以及人文的角度记录中国电竞。
电竞这个行业目前逐渐完善，现在通过建立代表城市、东西部阵营、设立青训营、取消降级、设立工资帽和最低工资等方式，让电竞的运营更加健康。
但是现在这个行业还是很辛苦，并不是外人看来打游戏轻松赚钱，由于这是个新兴行业，没有前车之鉴，每个电竞从业者（包括选手、解说、主播等）都是摸着石头过河，看到选手们半军事化训练，看到苏小妍手记各种数据为了能解说有话说，看到大仙说“每天都会有直播的压力，说不定哪天就凉了”。他们更加真实的一面确实触动到我了。
还有一点就是，电竞确实能够给人带来热血的感觉，希望这个行业发展的越来越好！
杀戮都市：O / GANTZ:O (2016)     2020-04-22 / 血腥 暴力 漫画改编 / ★★☆☆☆ 较差
